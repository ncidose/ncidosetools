# NCI Dose Tools Website Archive

This folder is the legacy website source that was formerly deployed from
`ncidose/ncidosetools`.

The current live documentation portal is maintained in the separate
[`ncidose/ncidose.github.io`](https://github.com/ncidose/ncidose.github.io)
repository and published at:

https://ncidose.github.io/

## Local Development

If this archive needs to be inspected locally:

```sh
npm i
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
